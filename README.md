#Drag & Drop Sample for Objective-C

This is a very easy sample of Drag & Drop app, and judge collision detection when its gesture is finished.

##Use
- UIView
- UIGestureRecognizer

##Environment
- Xcode5.0 and greater
- iOS7 and greater
- Objective-C
